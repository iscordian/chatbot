# Enplex.js chatbot
*Must star this repository*

## Guide

**Follow step by step**:

```bash
git clone https://github.com/iscordian/chatbot.git
cd chatbot
npm init -y
npm install enplex.js erislite
```

### Add you token in .env and run:

```bash
node bot.js
```
